Back up your Spotify playlists

This tool prompts you to log in with Spotify and then generates a backup of your playlists.

Generate backup

Permissions

The script will ask you for user-read-private and playlist-read-private permissions, which it needs to read the playlists for your user account.

What info is included?

To keep the file size down, I'm currently including basic fields like playlist name and URL, track name and URL, artist names and URLs, and album name and URL. There's lots more info available from the API but including it generates huge files.

What's excluded?

The only thing I know to be missing is collaborative playlists, as the Spotify API doesn't currently return these. If you need your collaborative playlists to be present in the backup, temporarily remove their collaborative status.

What's generated?

It generates one big JSON file that represents your playlists and the tracks within them.
